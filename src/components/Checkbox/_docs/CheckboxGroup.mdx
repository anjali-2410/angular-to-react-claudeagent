# CheckboxGroup

Checkbox groups allow users to select one or more options from a list. Unlike radio buttons, multiple selections are possible. They are commonly used in forms, settings, and filters where users need to select multiple items.

### Guidelines

Use checkbox groups when users need to select multiple options from a list. Checkbox groups allow users to see all available options and make multiple selections. They are ideal for settings, filters, and preference selections where multiple items can be selected.
