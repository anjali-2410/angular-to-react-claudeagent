# Checkbox

A checkbox is an input field that allows users to select one or more options from a list. Checkboxes have three selection states: unselected, selected, and indeterminate. The indeterminate state is used when a collection of sub-choices have both unselected and selected states.

### Guidelines

- Use checkboxes when users can select multiple options independently within a list.
- Avoid using checkboxes for mutually exclusive choices; use radio buttons instead.
- Keep checkbox labels short, clear, and placed to the right of the checkbox for readability.
- Use the indeterminate state to reflect a partially selected group of related options, not as a third persistent state.
- Ensure that each checkbox has an accessible name and that related checkboxes are grouped logically.

> **Important:** For validation errors, use `CheckboxGroup` rather than individual `Checkbox` components.
