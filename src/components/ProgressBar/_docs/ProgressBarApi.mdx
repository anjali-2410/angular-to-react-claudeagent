## Progress Bar: API

```ts
import { ProgressBar } from '@niceltd/sol-react/progress-bar';
```

### Props

| Name | Type | Default | Description |
| --- | --- | --- | --- |
| `progress` | `number` | `0` | The current progress value. Accepts 0–100. Values above 100 are capped at 100. |
| `ariaLabel` | `string` | `undefined` | Accessibility label for screen readers. Provides context about what the progress bar represents. |
| `ariaHidden` | `boolean` | `false` | When `true`, hides the progress bar from screen readers. |
