## Progress Bar

A determinate progress bar displays the completion status of a known task length. Use it when you can measure progress toward completion. Avoid showing indeterminate states with this component; use spinner or loading indicators for unknown durations.

### Guidelines
- Use a clear aria label to describe what is in progress (e.g., "File upload").
- Keep progress values between 0 and 100; values above 100 are capped.
- Place above or near the task context so users understand what is progressing.
- Avoid over-animating; stick to simple fills for accessibility.
