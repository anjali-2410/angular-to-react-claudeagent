## Dropdown

The `Dropdown` component is a versatile and flexible UI element that allows users to select one or more options from a list. It supports features such as search, select all, and custom templates for options, providing a robust selection experience for your project.

<hr/>

## Guidelines

The dropdown component can display a large number of items, but the maximum number of options rendered is controlled by the `maxItemsLimit` prop (default: **2500**; can be customized per use case). If the options array exceeds this value, only the first `maxItemsLimit` options are shown. This helps ensure optimal performance and user experience.

**Recommendations:**
1. **Minimal Display**: Show only as many items as necessary. Use the search functionality to help users quickly find and select items.
2. **Performance Considerations**: For very large datasets, set an appropriate `maxItemsLimit` and consider server-side search or pagination to avoid performance issues.
3. **Customization**: Adjust `maxItemsLimit` as needed for your use case. The default is 2500.
4. **Seek Expert Advice**: For complex scenarios, contact the **Avengers team & UX** for guidance.

### Sorting Options

**Features:**
- **Automatic Sorting**: Options are sorted by default in ascending alphabetical order
- **Custom Sort Key**: Sort by any property in your option objects, string or number fields
- **Sort Order Control**: Choose between ascending (`asc`) or descending (`desc`) order
- **Performance Optimized**: Sorting uses `useMemo` for efficient updates
