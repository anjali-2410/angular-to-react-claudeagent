# Switch

A Switch is a visual toggle control that can be turned on or off. Switches are used to enable or disable features, settings, or states in your application.

<hr/>

## Guidelines

- Use a Switch for binary on/off decisions that take effect immediately
- Always associate a descriptive label with every Switch using the `label` prop or `ariaLabel`
- Use the `symbol` prop to add check/dash icons inside the thumb for improved visual clarity
- Choose `size="small"` in compact UIs; `size="large"` (default) for standard contexts
- Mark required switches with `required={true}`
- Use `readonly={true}` when showing state that cannot be changed in the current context
