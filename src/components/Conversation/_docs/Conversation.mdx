## Conversational UI Guidance

SOL contains a number of components and helper classes that can be used to implement conversational UI designs into your application.

---

### Component Variants

### Prompt

The `ConversationPrompt` component offers users the ability to type text input and send messages. It includes a suggestion list from which users can select options.

```tsx
<ConversationPrompt
  placeholder="test"
  maxRows={4}
  showCharacterCount
  maxLength={200}
  suggestions={suggestions}
  showPixieIcon
  value={textInputValue}
  fetchSuggestions={fetchSuggestions}
  onActionButtonClicked={handleSubmit}
/>
```

---

### Suggestions array type

```ts
const suggestions = [
  { label: 'What are the top 10 call reasons?', tooltipText: 'Suggestion 1' },
  { label: 'Show me 10 agents with highest AHT?',  tooltipText: 'Suggestion 2' },
];
```

### Cards

To add a conversation with cards to your application, use the following components and helper classes.

#### `.sol-conversation`

All cards in a conversation need to be contained in a div with class `sol-conversation`:

```tsx
<div className="sol-conversation">
  {/* cards go here */}
</div>
```

#### `<Card>` with `.sol-conversation-question` / `.sol-conversation-response`

Wrap each `Card` in a div with one of these classes:

```tsx
<div className="sol-conversation">
  <div className="sol-conversation-question">
    <Card><CardBody>User's question text</CardBody></Card>
  </div>
  <div className="sol-conversation-response">
    <Card><CardBody>AI response text</CardBody></Card>
  </div>
</div>
```

#### `.sol-conversation-card-spinner`

Wrap a spinner inside the card body:

```tsx
<CardBody>
  <div className="sol-conversation-card-spinner">
    {/* spinner element */}
  </div>
</CardBody>
```

#### `.sol-conversation-card-button-block`

For follow-up buttons inside a card body:

```tsx
<CardBody>
  <div className="sol-conversation-card-button-block">
    <Button size="medium" variant="secondary">View as Chart</Button>
    <Button size="medium" variant="secondary">Another Action</Button>
  </div>
</CardBody>
```

#### `.sol-conversation-card-action-block` + `.sol-conversation-card-timestamp`

Inside a `CardFooter`:

```tsx
<CardFooter>
  <div className="sol-conversation-card-action-block">
    <Button size="small" variant="tertiary" icon={<SaveIcon />} />
    <Button size="small" variant="tertiary" icon={<MoreIcon />} />
  </div>
  <div className="sol-conversation-card-timestamp">10:17</div>
</CardFooter>
```

---

### Inputs / Outputs

- **Placeholder** — sets the placeholder text in the textarea
- **maxRows** — maximum rows before textarea scrolls (recommended: 4)
- **showCharacterCount** — displays character count
- **maxLength** — maximum character count
- **suggestions** — static list of suggestion items
- **fetchSuggestions** — async callback to fetch suggestions on demand
- **showPixieIcon** — shows/hides the AI (Pixie) icon button
- **value** — controlled textarea value
- **onActionButtonClicked** — emitted when submit is clicked (receives current value)
