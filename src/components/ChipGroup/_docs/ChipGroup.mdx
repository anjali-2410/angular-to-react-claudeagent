# Chip Group

The `ChipGroup` component is a versatile and flexible UI element that allows users to display a list of tags.

---

## Guidelines

- Use chip group for compact, removable tag collections where keyboard navigation is required.
- Keep labels concise and use overflow behaviour (`inline` or `popover`) when all tags cannot be shown inline.
- When chips are removable, keep `allowClearAll` enabled when users may need to quickly reset all tags.
- Chips in the group use the same keyboard and remove behaviour as `Chip`.
