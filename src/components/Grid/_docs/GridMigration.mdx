## Grid: Migration Guide

### Imports

**Angular:**
```ts
import { GridModule } from '@niceltd/sol/grid';
```

**React:**
```tsx
import { Grid } from '@niceltd/sol-react';
```

### Angular → React Migration

Replace the `sol-grid` element with the `<Grid>` React component and convert Angular bindings to React props:

<div className="container-flex-direction-row">
  <div className="col-left">

**Angular:**
```html
<sol-grid
  [columnDefs]="headerNames"
  [rowData]="rowData"
  [gridOptions]="gridOptions"
  [pagination]="pagination"
  [maxItemsPerPage]="pageSize"
  [pageNumber]="customPageNumber"
  [gridState]="gridState"
  (saveState)="saveGridState($event)"
  (changePageClick)="onPageChange($event)"
>
</sol-grid>
```

  </div>
  <div className="col-right">

**React:**
```tsx
<Grid
  columnDefs={headerNames}
  rowData={rowData}
  gridOptions={gridOptions}
  pagination={pagination}
  maxItemsPerPage={pageSize}
  pageNumber={customPageNumber}
  gridState={gridState}
  onSaveState={(state) => saveGridState(state)}
  onChangePageClick={(page) => onPageChange(page)}
/>
```

  </div>
</div>

### Changed Props

<table>
  <tr>
    <th>Angular Input</th>
    <th>React Prop</th>
    <th>Notes</th>
  </tr>
  <tr>
    <td>`[columnDefs]="col"`</td>
    <td>`columnDefs={col}`</td>
    <td>Same type: `(ColDef | ColGroupDef)[]`</td>
  </tr>
  <tr>
    <td>`[rowData]="data"`</td>
    <td>`rowData={data}`</td>
    <td>Same</td>
  </tr>
  <tr>
    <td>`[pagination]="true"`</td>
    <td>`pagination={true}`</td>
    <td>Same</td>
  </tr>
  <tr>
    <td>`[pageNumber]="n"` (signal input)</td>
    <td>`pageNumber={n}`</td>
    <td>Same — controls pagination externally (1-based)</td>
  </tr>
  <tr>
    <td>`(saveState)="fn($event)"`</td>
    <td>`onSaveState={(state) => fn(state)}`</td>
    <td>Renamed with `on` prefix</td>
  </tr>
  <tr>
    <td>`(saveSort)="fn($event)"`</td>
    <td>`onSaveSort={(sort) => fn(sort)}`</td>
    <td>Renamed with `on` prefix</td>
  </tr>
  <tr>
    <td>`(changePageClick)="fn($event)"`</td>
    <td>`onChangePageClick={(page) => fn(page)}`</td>
    <td>Renamed with `on` prefix</td>
  </tr>
  <tr>
    <td>`[frameworkComponents]="comps"`</td>
    <td>`frameworkComponents={comps}`</td>
    <td>Same — pass React components, not Angular components</td>
  </tr>
  <tr>
    <td>TranslationService injected</td>
    <td>`i18n={...}` prop</td>
    <td>Pass translated strings via the `i18n` prop object</td>
  </tr>
</table>

### Migrate from Breeze

The SOL React Grid should act and behave like the Breeze Grid with the following changes:

<table>
  <tr>
    <th>Breeze Input</th>
    <th>SOL React Prop</th>
    <th>Notes</th>
  </tr>
  <tr>
    <td>usePagination</td>
    <td>pagination</td>
    <td>(renamed)</td>
  </tr>
  <tr>
    <td>suppressCellFocus</td>
    <td>—</td>
    <td>(removed) Cell focus is always enabled for keyboard accessibility compliance</td>
  </tr>
</table>
