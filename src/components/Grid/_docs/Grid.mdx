## Grid

The `Grid` component is a flexible component for displaying data in a tabular format. It leverages the capabilities of the renowned ag-Grid library, providing a rich set of features for complex data manipulation and visualization.

### Guidelines

Internally, `Grid` uses `ag-Grid React` to create a highly customizable grid structure. It abstracts away the complexity of ag-Grid, providing a simpler interface for developers while still maintaining the extensive functionality of ag-Grid. This includes features like sorting, filtering, grouping, and much more.

<hr/>

### Component Variants

**Import the Grid component**:

```tsx
import { Grid } from '@niceltd/sol-react';
```

Here is an example of how to use the `Grid` component in React:

## Grid Example:

```tsx
<Grid
  columnDefs={headerNames}
  rowData={rowData}
  gridOptions={gridOptions}
  pagination={true}
  maxItemsPerPage={pageSize}
  onSaveState={(state) => console.log('Save Grid state', state)}
/>
```

```tsx
const gridOptions: GridOptions = {
  rowSelection: 'multiple',
  onGridReady: (params) => {
    params.api.sizeColumnsToFit();
  },
};
```
