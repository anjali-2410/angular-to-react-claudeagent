import { ActionBar } from '@niceltd/sol-react/action-bar';

## Migration to v16

Breaking input name changes from SOL v14 → v16 (snake_case → camelCase):

| v14 input | v16 / React prop |
| --- | --- |
| `show_action_bar` | `showActionBar` |
| `action_bar_text_content` | `actionBarTextContent` |
| `button_icon` | `buttonIcon` |
| `button_text` | `buttonText` |
| `aria-label` | `ariaLabel` |

---

## Angular → React Migration

| Angular `@Input` / `@Output` | React prop |
| --- | --- |
| `[showActionBar]` | `showActionBar` |
| `[actionBarTextContent]` | `actionBarTextContent` |
| `[buttonIcon]` | `buttonIcon` |
| `[buttonText]` | `buttonText` |
| `[styleData]` | `styleData` |
| `[iconPath]` | `iconPath` |
| `[ariaLabel]` | `ariaLabel` |
| `[liveMessage]` | `liveMessage` |
| `(buttonClicked)` | `onButtonClicked` |

```tsx
// Angular
<sol-action-bar
  [showActionBar]="true"
  [actionBarTextContent]="'Impersonating user'"
  [buttonText]="'Stop'"
  (buttonClicked)="onStop()"
/>

// React
<ActionBar
  showActionBar
  actionBarTextContent="Impersonating user"
  buttonText="Stop"
  onButtonClicked={onStop}
/>
```
