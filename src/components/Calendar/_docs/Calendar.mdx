# Calendar

The Calendar component provides a versatile date selection interface. It supports multiple selection modes including single date, multiple dates, date ranges, and week-based selections.

---

## Guidelines

Use the Calendar component when:

- Users need to select one or more dates
- You want to provide a visual calendar interface
- Date range or week-based selections are required
- You need to highlight special dates or disable specific days

**Best Practices:**

- Choose the appropriate `type` based on your use case
- Use `specialDates` to highlight important events
- Use `daysDisabled` when certain days-of-week should not be selectable
- Set `min`/`max` to constrain the selectable date range
