# Autocomplete: API

## Component API Description

### Autocomplete

#### Props

<table>
	<tr>
		<th>Name</th>
		<th>Data Type</th>
		<th>Description</th>
	</tr>
	<tr>
		<td>minimumSearchTextLength</td>
		<td>number</td>
		<td>The minimum number of characters required before showing autocomplete suggestions. Default: 3</td>
	</tr>
	<tr>
		<td>placeholder</td>
		<td>string | undefined</td>
		<td>Placeholder text displayed in the input field. Default: 'Search for item'.</td>
	</tr>
	<tr>
		<td>ariaLabel</td>
		<td>string</td>
		<td>ARIA label for accessibility. Required for screen reader support.</td>
	</tr>
	<tr>
		<td>panelId</td>
		<td>string | undefined</td>
		<td>ID for the dropdown panel. Auto-generated if omitted.</td>
	</tr>
	<tr>
		<td>value</td>
		<td>string | undefined</td>
		<td>Controlled value for the search input.</td>
	</tr>
	<tr>
		<td>onChange</td>
		<td>(value: string) =&gt; void</td>
		<td>Called whenever the input text changes.</td>
	</tr>
	<tr>
		<td>onOptionSelected</td>
		<td>(event: AutocompleteOptionSelectedEvent) =&gt; void</td>
		<td>Called when an option is selected from the dropdown.</td>
	</tr>
	<tr>
		<td>onClearedSearchText</td>
		<td>() =&gt; void</td>
		<td>Called when the clear button is clicked to remove search text.</td>
	</tr>
	<tr>
		<td>onDropdownVisible</td>
		<td>(visible: boolean) =&gt; void</td>
		<td>Called when the dropdown opens (true) or closes (false).</td>
	</tr>
	<tr>
		<td>children</td>
		<td>ReactNode</td>
		<td>AutocompleteOption children to display in the dropdown.</td>
	</tr>
</table>
